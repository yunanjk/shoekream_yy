# 프로젝트 상세 내용 

## 기능
- 로그인 및 회원가입
- 경매 기반 거래 시스템
- 구매/판매
- 체결내역 기반 시세 그래프
- 한정판 제품 추첨 이벤트
- 세분화된 필터
- 상품 검색
- SNS 기능 스타일 공유 (팔로우, 팔로워, 게시물 등록, 댓글, 좋아요, 해시태그)
- 관심 상품
- 카드 정보, 계좌 정보, 주소, 프로필 수정
- 회원관리 (블랙리스트/패널티)
- 문의 및 공지사항
